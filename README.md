WHAT DOES THE COW SAY?

usage:
```javascript
console.cowsay(text [, line_length=50]);
```

example:
```javascript
console.cowsay("Lorem ipsum dolor sit amet, consectetur adipiscing elit");
```

output:
```
  ----------------------------------------
< Lorem ipsum dolor sit amet, consectetur  >
< adipiscing elit                          >
  ----------------------------------------
         \   ^__^
          \  (oo)\________
             (__)\        )/\
                 ||----w |
                 ||     ||
```

```javascript
console.cowsay("Lorem ipsum dolor sit amet, consectetur adipiscing elit", 100);
```
